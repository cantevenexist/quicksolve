### Шаг 1 – миграция:
Docker производит копирование (наследование) миграций баз данных моделей Django.
Для первого запуска воспользуйтесь следующей командой для создания инициализирумых миграций:
'''
python3 djangoapp/manage.py makemigrations
'''

### Шаг 2 – сборка контейнеров Docker:
Для запуска воспльзуйтесь следующей командой:
'''
docker-compose -f docker-compose.yml up --build
'''
